# Grape Leaf Disease Detection using Deep Learning

This project focuses on detecting diseases in grape leaves using deep learning techniques. It leverages convolutional neural networks (CNNs) to classify images of grape leaves into healthy or various disease categories, helping farmers and agriculturists to identify and manage plant diseases effectively.
